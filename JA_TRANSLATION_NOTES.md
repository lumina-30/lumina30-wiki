# 日本語版 翻訳判断メモ

このメモはPoCの保守用です。概念の新しい定義を作るものではありません。

| English term | 日本語表現 | 判断 |
|---|---|---|
| Effective Human Refusal | 実効的人間拒否 | LUMINA-30中核語。初出・ページタイトルでは英語を併記。形式的監督ではなく、不可逆化前に実際に効力を持つ拒否を指す。 |
| Irreversible Escalation | 不可逆化／不可逆的エスカレーション | 日本語の読みやすさを優先して「不可逆化」を主表記とし、初出で英語を併記。単なる被害発生後ではなく、拒否・停止・検証・修正が実質的に効かなくなる境界を含む。 |
| Stop Authority | 停止権限 | 技術的な停止機能だけでなく、正当な人間・制度が適時に行使できる権限を含む。 |
| Pre-Incident Boundary Review | 事故前境界レビュー | 一般的な導入前審査の代替ではなく、不可逆化前の拒否有効性を追加確認する層。 |
| One-Question Pilot | 一問パイロット | 固有の実務入口として英語を併記。完全なリスク評価・認証チェックリストではない。 |
| Human Control | 人間による制御 | 一般的なAIガバナンス概念。LUMINA-30固有概念として再定義しない。 |
| Human Override | 人間によるオーバーライド | 「人間による上書き」では意味が狭くなるため原語に近い表現を採用。名目上の機能と実効的拒否を区別。 |
| Evidence Preservation | 証拠保全 | ログ保存だけに限定せず、承認・権限・時刻・拒否経路を再構成できる証拠を含む。 |
| Reversible Prosperity Path | 可逆的繁栄の道 | LUMINA-30の代替名ではなく、境界条件が保たれるときの肯定側の進路。 |
| non-binding | 非拘束 | 法的効力、認証、採用、承認、安全保証を生じさせない位置付けを保持。 |
